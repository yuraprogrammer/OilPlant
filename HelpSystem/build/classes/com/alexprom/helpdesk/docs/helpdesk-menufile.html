Описание меню "Файл"




Меню "Файл" предназначено для выполнения операций с актом.

Перечень операций, производимых с актом:

  • Создать акт
  • Открыть акт
  • Закрыть акт
  • Сохранить акт
  • Печать акта


Команда меню "Выход" служит для выполнения операции выхода из приложения.
