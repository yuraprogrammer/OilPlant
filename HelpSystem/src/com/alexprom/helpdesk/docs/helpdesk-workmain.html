Описание области "Данные по переработке"



Область данных по переработке можно функционально разделить на следующие подобласти:

  • Переработка сырья
  • Производство БЛФ
  • Производство АкДГ
  • Производство ОТГ
  • Выкачка из дренажной емкости
  • Показания счетчика подпитки воды
  • Сдренировано воды
