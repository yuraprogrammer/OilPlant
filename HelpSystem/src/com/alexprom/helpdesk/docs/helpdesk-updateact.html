Описание операции "Обновить акт"



Если параллельно с пользователем другой пользователь на другом компьютере осуществлял операции по редактированию акта, то эти изменения не
будут видны текущему пользователю.

Если такая ситуация возникла, то для отображения измененных другим пользователем данных, служит операция "Обновить акт".

При вызове данной команды, все поля акта, измененные и сохраненные другим пользователем будут изменены.

Если параллельно данные акта не менялись и не сохранялись, то при вызове команды "Обновить акт", все поля акта остануться неизменными.
Если текущий пользователь перед сохранением акта не выполнял обновление, а параллельно другой пользователь вносил изменения и сохранил их,
то в акт сохраняться те данные, которые были сохранены последними.
