Описание области "Сдренировано воды"



Область ввода данных о количестве сдренированной воды выглядит следующим образом:



Данные о количестве сдренированной воды из сырья и из БЛФ вводятся вручную пользователем.

Введенные данные используются при расчете процента выхода продукта за вычетом количества сдренированной воды из сырья и БЛФ.
