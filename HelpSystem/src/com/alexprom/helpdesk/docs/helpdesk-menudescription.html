Главное меню приложения

Главное меню приложения состоит из следующих пунктов меню:
  • Файл
  • Правка
  • Вид
  • Переход
  • Сервис
  • Окно
  • Справка

Панель управления содержит команды быстрого доступа к операциям:
  • Создать акт
  • Открыть акт
  • Сохранить акт
  • Печать акта
  • Обновить акт
